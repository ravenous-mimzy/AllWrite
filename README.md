# Version 0.0.1

Electron-based writer's app (Characters module) with List, Cards, and Importance views.

## Highlights
- Unified button-based sort and per-importance/per-gender filters across views
- Importance view sorts alphabetically within each group
- Shared sort/filter state persists across views and section changes
- Editor panel integration and layout management

## Run
```
npm start
```

## Tech
- Electron
- Plain JavaScript + DOM APIs
- CSS for styling
